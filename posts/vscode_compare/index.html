[VSCode] VSCode에서 파일/소스 비교하기(Source Compare)

kkangz Jul 132021-07-13T17:30:00+09:00
Aug 142021-08-14T01:11:22+09:00 1 min

VSCode 에서 간단 파일 비교를 위한 팁 입니다. 먼저 Compare 창 진입은 VSCode 내에 command로 하는데, 방법은 아래와 단축키로 Show all command 창에서 명령을 내립니다.

[Windows] Ctrl + Shift + p

[Mac] Command + Shift + P or F1 (with fn key)

위 단축키를 누르면 상단에 command 입력창이 뜨는데, 요기에 compare 만 치시면 아래와 같이 3개 명령이 나옵니다.

그리고 각 편한 방법대로 비교하시면 됩니다!

  1. Compare Active File With.. (현재 열려 있는 파일과, 다른 파일간에 비교하기) 다른 파일을 선택하면 이렇게 파일 비교를 해 준다.

  2. Compare Active File with Clipboard. (현재 열려있는 창과, 클립보드 내용과 비교하기) 현재 Clipboard 에 복사되어있는 코드랑 비교할때 사용.

  3. Compare Active File with saved. 현재 수정된 코드와 저장전 코드와 비교하기 작업중인 문서가 저장하기 전 이라면, 저장 전과 현재 코드를 비교 해줌.

이상입니다!
